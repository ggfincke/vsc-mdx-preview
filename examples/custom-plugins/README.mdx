## Requirements

This example requires **Trusted Mode**:

1. Trust this workspace
2. Enable scripts: `mdx-preview.preview.enableScripts: true`
3. Install dependencies: `npm install`

## Setup

```bash
npm install
```

Then open this file in MDX Preview (Cmd+K X / Ctrl+K X).

## How It Works

The `.mdx-previewrc.json` configuration file tells MDX Preview to:

1. Load custom remark/rehype plugins from `node_modules`
2. Auto-import the `Callout` component from `./src/components/Callout.tsx`
3. Apply these transformations before rendering

This keeps your MDX files clean while still using custom functionality!
